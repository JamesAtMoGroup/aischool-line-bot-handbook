01 · 介紹

它是什麼

它是學院的 LINE 學習小幫手，24 小時在線。
你想學什麼、卡在哪、看到錯誤訊息看不懂 — 都可以問它。它認識學院所有課程，會幫你找到對的那一集。

02 · 對話示範

四個常見情境

下面四個例子，左邊是情境介紹 + 提問樣板，右邊是你會在 LINE 看到的實際對話樣子。點「複製」直接拿樣板來用。

情境 一

想知道學院有沒有教某題

你想找一個特定主題，看學院內有沒有相關內容、該從哪集開始。把「想學的原因」跟「已有基礎」一起寫，它推得才精準。

提問樣板
我想學：[主題] 我之所以想學是因為：[實際應用 / 想解決什麼] 我目前已經會：[已有的基礎] 學院有哪幾集在教？推薦我從哪個開始？
AI
學習小幫手
我想學：n8n 自動化。
之所以想學：把每週幫客戶整理名片的時間從 3 小時砍到 30 分。
已經會：基本的 Excel + 簡單 Zapier。
學院有哪幾集在教？
你的場景很適合直接看「n8n 名片管家」（4 集），完成後就有現成名片自動入庫流程，跟你的需求對到。如果還想學工作流整合更深，再進「n8n 高效自動化」。要列每集內容嗎？
情境 二

剛加入不知道從哪開始

你還沒決定該先看哪一支課。背景、目標、現有基礎、可投入時間都寫清楚，它才能規劃符合你節奏的順序。

提問樣板
我的背景：[現在職業 + 公司性質] 我想 6 個月後做到：[具體可量化的目標] 我現在會用的 AI 工具：[列舉] 我每週可學的時間：[小時數] 從哪堂課開始最合適？
AI
學習小幫手
背景：行銷企劃，中小企業 30 人。
6 個月後想做到：每月幫公司產 20 篇社群內容、品質不掉。
會的工具：ChatGPT 偶爾用。
每週可學：5 小時。
從哪堂開始？
先從「Claude 實戰」前 3 集打底，把對話跟 prompt 邏輯練熟（約 2-3 週）。穩了之後接「Vibe Coding」第 1 集做出第一個小成品，給你信心。月底再聊一次，我幫你看是不是要加進 n8n。
情境 三

卡關時 — 用「描述四元素」

提問越具體，AI 越能精準幫你定位 — 多花 30 秒把這 4 個欄位填清楚（想做、做了、期待、實際得到），問題就能解得更快。

提問樣板（描述四元素）
我想做：[想達成什麼具體結果] 我做了：[實際步驟 + 用了哪些工具] 我期待：[原本以為會看到什麼] 實際得到：[真正發生的狀況 / 錯誤訊息]
AI
學習小幫手
我想做：把 Vibe Coding 第三集的網站部署上線。

我做了：照影片把 code 推到 GitHub，Zeabur 連 repo 按部署。

我期待：跟影片一樣顯示首頁。

實際得到：頁面跑出來但空白，console 顯示「Module not found: react-router」。
描述很清楚，問題定位很快 — 你少裝 react-router。影片裡有提到那一步用 npm install react-router-dom，你可能跳過了。在 GitHub repo 跑一次 npm install react-router-dom，commit 推上去 Zeabur 會自動重 deploy。試完跟我說結果。

04 · 能做什麼

它可以幫你做這些事

下面六件事是它的專長。其他像學費、約老師、改個資這類事情，第 05 段「找誰處理」會告訴你該找誰，會更快。

找對應課程
幫你找學院內哪一集、哪一章在教某個主題，給直達連結。不會推薦外部影片。
講解概念
用白話解釋專有名詞，但會告訴你哪一集教得更完整，鼓勵你看完整課程加深。
推薦工具
從學院策展過的工具庫推薦 1–3 個適合的 AI 工具，不導去外部不熟悉的服務。
看圖回答
你傳的截圖、錯誤訊息、做出的成品它都看得到並回答。

建議先用文字把問題講清楚，再附上圖片當輔助 — 它就能一秒抓到你要問的重點。

圖片用電腦截圖（Mac 按 Shift + Cmd + 4、Windows 按 PrintScreen）或直接存原始的錯誤訊息圖檔效果最好。手機翻拍電腦螢幕容易拍到反光跟桌面雜物、畫面也比較糊，會讓它看不清楚你的問題。
查最近文章
問它「最近有發 X 主題的文章嗎」，它從最近 30 篇學院文章內找。
查學習歷程
綁定 email 後，它知道你的諮詢紀錄、目前能力等級、業師建議的方向。

使用方式：LINE 開著就能問，24 小時不限次數。
